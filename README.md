# Zombie Defense Game
## Basic keys
W : move up\
A : move left\
D : move right\
S : move down\
mouse left-click : fire gun

## Goal
Eliminate all the zombie waves to survive!

## How to run
Go to /src and open main.cpp, then compile it.
